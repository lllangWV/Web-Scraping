Common Supergroups of two Space Groups


Common Supergroups
COMMONSUPER calculates the common supergroups of two space groups given:

  • Space groups numbers
  • Number of formula units (per conventional unit cell)
  • A maximum k-index which defines the maximum cell multiplication.

Only the default choice for the conventional setting of the space groups is used.

Please, enter the sequential number of the space group 1 as given in International Tables for Crystallography, Vol. A or choose it: Z1 (conventional)
Please, enter the sequential number of the space group 2 as given in International Tables for Crystallography, Vol. A or choose it: Z2 (conventional)
Maximum k-index
NOTE: the program uses the default choice for the group setting.
