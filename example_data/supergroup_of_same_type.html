Each supergroup is determined by:
  • The transformation matrix relating the basis of the supergroup with that of the subgroup - given in the column Transformation matrix.
  • The coset representatives in the decomposition of the supergroup with respect to the subgroup G = H + Hg + ... - one representative from each coset of the decomposition is given in the column Coset representatives. To see all of the representatives click over [Full cosets].

There are group-supergroup pairs for which there are parameter/s in the translational part of the transformation matrix. In this case if there are the same parameters in the coset representatives for each value of the parameters different supergroup of the same type will be obtained.
